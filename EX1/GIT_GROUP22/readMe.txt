Conversion Calculator

## Description
This is a simple conversion calculator for converting numbers between decimal and hexadecimal bases. The program provides options to convert in both directions: decimal to hexadecimal and hexadecimal to decimal. 

## Features
- Convert from decimal to hexadecimal
- Convert from hexadecimal to decimal
- Simple and interactive command line interface

## Requirements
- Python 3.12.3

## Functions
### `decimal_to_hexadecimal(dec_num)`
Converts a decimal number to its hexadecimal representation.

### `hexadecimal_to_decimal(hex_num)`
Converts a hexadecimal number to its decimal representation.

### `is_hexadecimal(input_str)`
Checks if the given input string is a valid hexadecimal number.

### `is_valid_decimal(input_str)`
Checks if the given input string is a valid decimal number.

### `convert_numbers()`
A calculator function that provides an interactive command line interface for conversions.

## Example Usage
```sh
--- Conversion calculator between bases ---
1: Conversion from decimal to hexadecimal base
2: Conversion from hexadecimal to decimal base
-1: EXIT

Select your choice : 1
Enter a decimal number to convert: 255
255 in decimal is FF in hexadecimal.

Select your choice : 2
Enter a hexadecimal number to convert: FF
FF in hexadecimal is 255 in decimal.
```
