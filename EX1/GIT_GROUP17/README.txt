ברוכים הבאים למחשבון להמרת מספרים בין בסיסים עשרוני להקסדצימלי וההפך!
המחשבון מאפשר לבצע המרה מהירה ונוחה בין שני הבסיסים.

שימוש במחשבון:
1. הפעל את התוכנית.
2. בתפריט הראשי תוצג לך רשימה של שלוש אפשרויות:
	1. המרת מספר עשרוני (Decimal) למספר הקסדצימלי (Hexadecimal).
	2. המרת מספר הקסדצימלי (Hexadecimal) למספר עשרוני (Decimal).
	3. יציאה מהמערכת.
3. בחר את האפשרות הרצויה באמצעות הזנת המספר המתאים.

עבור אפשרות 1:
	1. הזן מספר עשרוני חוקי.
	2. אם הקלט אינו תקין (למשל, תו שאינו מספרי), תוצג הודעת שגיאה.
	3. המערכת תבצע את ההמרה ותציג את התוצאה.

עבור אפשרות 2:
	1. הזן מספר הקסדצימלי חוקי (תווים A-F ו-0-9 בלבד).
	2. אם הקלט אינו תקין (למשל, תו שאינו חלק מבסיס 16), תוצג הודעת שגיאה.
	3. המערכת תבצע את ההמרה ותציג את התוצאה.

עבור אפשרות 3:
	1. יציאה מהמערכת.

4. לאחר ההמרה המערכת תחזור באופן אוטומטי לתפריט הראשי כדי לבצע המרה נוספת או לצאת.

תכונות נוספות:
	1. המערכת תומכת גם באותיות קטנות (a-f) וגם באותיות גדולות (A-F) בקלט הקסדצימלי.
	2. אפס (0) יומר כראוי לכל בסיס.
	3. המערכת מתמודדת עם שבר עשרוני.
	4. המערת מתמודדת עם מספרים שליליים.
